yii2-sms
========

# Что это?

Это — набор компонентов для Yii2, упрощающий работу с различными сервисами рассылки СМС-сообщений.

# Как работает?

Подключается как обычный компонент и реалиует интерфейс SMS, который имеет метод `send(<to>,<text>)`

# Какие сервисы поддерживаются?

* GoldenTelecom

# Как использовать?

Добавьте в секцию `require` файла `composer.json` строку:
```
	"muvo/yii2-sms": "*"
```

Выполните `composer update`

В файле конфигурации подключите компонент нужного SMS-провайдера, например:
```
	'sms' => [
		'class' => '\muvo\yii\sms\providers\GoldenTelecom',
		'username' => <LOGIN>,
		'password' => <PASSWORD>,
	]
```

Теперь компонент доступен для использования! Чтобы отправить сообщение, используйте конструкцию вида `\Yii::$app->sms->send('79998887766','hello, world!')`

# PROFIT!
